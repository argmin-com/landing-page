# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** [contact@argmin.co](mailto:contact@argmin.co)

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for confirmed vulnerabilities.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest on `main` | Yes |

## Security Measures

- **Dependency scanning:** Dependabot monitors for vulnerable dependencies weekly
- **Code analysis:** CodeQL scans run on every push and PR to `main`
- **Dependency review:** PRs that modify dependencies are checked for known vulnerabilities
- **Lighthouse audits:** Performance, accessibility, best-practices, and SEO are validated in CI

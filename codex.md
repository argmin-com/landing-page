# Codex Entry Guide

This repository supports multiple coding assistants. Codex should use the shared repository guidance in [docs/assistant-guide.md](./docs/assistant-guide.md) instead of relying on a Codex-only rule set.

Before making changes:

1. Read [docs/assistant-guide.md](./docs/assistant-guide.md).
2. Read [README.md](./README.md) for project and deployment context.
3. Read the files you plan to modify and their direct dependencies.
4. Run the relevant validation commands before finishing work.

If guidance in this file and `.github/copilot-instructions.md` ever diverges, resolve the drift by updating both entrypoint files to match the shared assistant guide.
